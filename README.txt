TarefasApp — Pacote de Arquivos (adicione ao seu projeto existente)

Como usar:
1) Abra seu projeto MAUI 'TarefasApp' no Visual Studio (já criado com `dotnet new maui -n TarefasApp`).
2) Copie o conteúdo destas pastas/arquivos para as mesmas localizações no projeto:
   - App.xaml (SUBSTITUIR)
   - App.xaml.cs (SUBSTITUIR)
   - AppShell.xaml (SUBSTITUIR)
   - AppShell.xaml.cs (SUBSTITUIR)
   - Pages/MainPage.xaml (SUBSTITUIR)
   - Pages/MainPage.xaml.cs (SUBSTITUIR)
   - Pages/TaskDetailPage.xaml (NOVO)
   - Pages/TaskDetailPage.xaml.cs (NOVO)
   - Pages/AddEditTaskPage.xaml (NOVO)
   - Pages/AddEditTaskPage.xaml.cs (NOVO)
   - Models/TaskItem.cs (NOVO)
   - Services/TaskRepository.cs (NOVO)
3) No Visual Studio, clique com o botão direito no projeto > "Add > Existing Item..." e adicione os arquivos NOVOS, se não aparecerem automaticamente.
4) Compile e rode.

Obs.: Não incluímos a pasta Platforms nem o .csproj; use os do seu projeto.
